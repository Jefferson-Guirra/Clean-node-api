## Technologies used
- Nodejs
- Express
- Typescript
- Mongodb
- Bcrypt
- Fast-glob


## Instalation

Download the repository

```bash
$ git clone https://github.com/Jefferson-Guirra/Clean-node-api.git

```
Access the project folder

```bash
cd Clean-node-api
```
Install back-end dependencies

```bash
$ npm run install

```
## running app

### development
```bash
$ npm start
```

### watch mode
```bash
$ npm run start:dev
```

## Test

### Unit Test
```bash
$ npm run test:unit
```
### Integration Test
```bash
$ npm run test:integration
```
### Coverage Test
```bash
$ npm run test:ci
```
